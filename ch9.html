Chapter 9 : Finding Groups of Data Clustering with K-means

Useful cases:

1. Segmenting customers into groups with similar demographics or buying patterns for targeted  marketing campaigns

2. Detecting anomalous behavior, such as unauthorized network intrusions, by identifying patterns of use falling outside the known clusters.

3. Simplifying extremely large datasets by grouping features with similar values into a smaller number of homogeneous categories.

Key Point 1:

The image analysis a clustering problem or classification problem. Since clustering is a unsupervised machine learning model, it can classifies unlabeled examples. One of the most difficult problems in medical image analysis is to get images labeled by the physicians. If it’s done by manually, it will be costing and time-consuming.  Maybe we can use clustering algorithms first to label images and then use classification algorithms to solve the problem.

Instead of using CNN (convolutional neural network ) solely for medical image analysis, it could be a semi-supervised learning model. 1st step: using clustering algorithms to label images, then 2nd, using decision trees or other classification algorithms to classify images.

Key Point 2:

The process of finding the significant features are done manually and complicated. After list all features with their data, you have to manually find out the biggest number and the most significant features manually. This is tedious and easy to cause errors. There should be a function to find them. Maybe a sort function to analyze the data.
