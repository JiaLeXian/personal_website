Black Box Methods- Neural Networks and Support Vector Machines

An Artificial Neural Network(ANN) models the relationship between a set of input signals and an output signal using a model derived from our understanding of how a biological brain responds to stimuli from sensory inputs. ANN uses a network of artificial neurons or nodes to solve learning problems.

Turing Test

. Proposed in 1950 by Alan Turing, grades a machine as intelligent if a human being cannot distinguish its behavior from a living creatures.

Uses:

. Best apply to problems where the input data and output data are well-defined or at least fairly simple, yet the process that relates the input to output is extremely complex

Three Characteristics of Neural Network

. An activation function, which transforms a neuron’s combined input signals into a single output signal to be broadcasted further in the network

. A network topology (or architecture), which describes the number of neurons in the models as well as the number of layers and manner in which they are connected.

. The training algorithms that specifies how connection weights are set in order to inhibit or excite neurons in proportion to the input signal.

Activation Functions

Process incoming information and pass it through the network. A threshold activation function, as it results in an output signal only once a specified input threshold has been attained. Rarely used in real world.

Sigmoid Activation Function

Can fall anywhere between 0 to 1.

Is differentiable. It is possible to calculate the derivative across the entire range of inputs.

Sigmoid activation function is sometimes called squashing functions.

Different activation function serves different dataset. They key difference is the range of output signal.

The solution to squashing is to transform all neural network inputs such that the features’ values fall within a small range around 0.

Network Topology

The number of layers

Whether information in the network is allowed to travel backward.

The number of nodes within each layer of the network.

Deep Neural Network(DNN)

Multiple hidden layers is called Deep Neural Network and the practice of training such network is sometimes referred to as deep learning.

The direction of information travel

The addition of a short-term memory, or delay, increases the power of recurrent networks immensely. However, rarely used in the real world. Could be used in the stock market prediction, speech comprehension, or weather forecasting.
