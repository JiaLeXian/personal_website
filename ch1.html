Chapter 1 & Chapter 2: Introduction Of Machine Learning

Data has been in our life for a long time. Our brain is a super computer itself.

Machine learning focuses on teaching computers how to use data to solve a problem, while data mining focuses on teaching computers to identify patterns that humans then use to solve a problem. （Lantz, Brett. Machine Learning with R - Second Edition - Deliver Data Insights with R and Predictive Analytics (p. 3). Packt Publishing. Kindle Edition.）

Basic machine learning algorithms that are employed to teach computers to perform these tasks. （Lantz, Brett. Machine Learning with R - Second Edition - Deliver Data Insights with R and Predictive Analytics (p. 5). Packt Publishing. Kindle Edition.）

Heuristics, which are educated guesses about where to find the most useful inferences. （Lantz, Brett. Machine Learning with R - Second Edition - Deliver Data Insights with R and Predictive Analytics (p. 13). Packt Publishing. Kindle Edition.）

Supervised and unsupervised learning. Supervised learning is mostly used for predictive model and classification. In supervised model, model is given a clear path to learn and clear evaluation of how well the model performs. Examples of supervised model: classify spam in emails and predict housing price in certain areas. Linear regression and logic trees can be used in supervised model.

Unsupervised learning. Unsupervised learning is a descriptive model. No target to learn.Example: market basket analysis, pattern discovery and clustering.

I found most of problems in real world can be analyzed by supervised model.
